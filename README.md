## Getting started
1. Clone the repository
2. Install [PNPM](https://pnpm.io/installation)
3. ```shell
   pnpm i
   pnpm tauri
   ```

### To build the app in debug mode:
```shell
vite build
   ```

### To build the executable:
```shell
tauri build
```
